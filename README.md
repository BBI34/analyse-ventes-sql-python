# analyse-ventes-sql-python
Analyse des ventes sur 20 jours à l’aide de SQL et Python (Pandas &amp; Plotly) afin d’explorer le chiffre d’affaires, les performances produits et la répartition géographique des ventes.
# 📊 Analyse des ventes - SQL & Python

## Description du projet
Analyse des ventes d’une PME sur 20 jours afin d’étudier le chiffre d’affaires, les produits et les régions les plus performants.

## Objectifs
- Calculer le chiffre d’affaires total
- Analyser les ventes par produit
- Analyser les ventes par région
- Visualiser les résultats avec Python

## Données
- date
- produit
- prix
- qte
- region

## Outils utilisés
- SQL (SQLite)
- Python
- Pandas
- Plotly

## Résultats principaux
- Produit A = plus gros volume de ventes
- Produit C = plus rentable
- Région Sud = forte performance en CA

## Conclusion
Ce projet permet d’identifier les produits et régions les plus performants pour aider à la prise de décision.
